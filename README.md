#Intent-Level Hate News Classification using BiLSTM
📌 Project Overview
With the rapid surge of digital news, the spread of hateful content has become a significant social challenge. This project implements a Deep Learning framework to detect and classify news articles based on their intent. Unlike binary classifiers, this system categorizes news into specific sub-types of hate speech to provide more granular content moderation.

We utilize a Bidirectional LSTM (BiLSTM) network, which is highly effective for text classification as it captures context from both past and future states in a sequence, allowing the model to understand the subtle "intent" behind complex news sentences.
